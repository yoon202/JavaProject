# 📅 출석 관리 시스템 (Attendance Management System)

> **동의과학대학교 컴퓨터소프트웨어과** > **개발자:** 최민재 (202520509)  
> **개발 기간:** 2025.01 ~ 2025.02

Java를 활용하여 기존 출결 시스템의 UI 불편함을 개선하고, 사용자 편의성을 극대화한 학사 관리 솔루션입니다. 최소한의 클릭으로 데이터를 관리할 수 있도록 설계되었습니다.

## ✨ Key Features (주요 기능)

### 1. 인증 및 권한 시스템
* **권한별 차등 제공:** `ADMIN`(관리자)과 `USER`(일반 사용자) 권한 구분
* **관리자 모드:** 학생/교수/강의 정보 등록, 수정, 삭제 및 전체 권한
* **사용자 모드:** 배정된 강의 조회 및 출결 현황 확인

### 2. 효율적인 정보 관리
* **학생/교수 관리:** 학번과 내부 ID를 구분하여 데이터 무결성 유지
* **강의 통합 관리:** 교수별 강의 배정 및 통합 조회 시스템
* **일관된 UI/UX:** 모든 관리 탭에서 통일된 디자인을 제공하여 학습 비용 최소화

### 3. 스마트 출결 시스템
* **맞춤형 검색:** 날짜별, 학생별 필터링 기능을 통한 빠른 데이터 조회
* **데이터 보존:** 학생 정보(이름, 학번 등)가 변경되어도 기존 출결 기록 유지

---

## 🗄 Database Structure (DB 구조)

데이터베이스는 데이터 간의 관계를 최적화하여 설계되었습니다.



* **users**: 로그인 계정 정보 및 권한(Role) 관리
* **students**: 학생 인적 사항 (student_id와 내부 고유 id 구분)
* **professors**: 교수 정보 관리
* **courses**: 강의 정보 및 교수 연결 (`professor_id`)
* **attendance**: 학생별 출결 기록 (`student_id` 외래키 연결)
